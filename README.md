# Portfolio V2

A modern, responsive portfolio website showcasing web development projects and technical skills.

## About

Personal portfolio built with vanilla HTML, CSS, and JavaScript featuring smooth animations, responsive design, and clean UI. Highlights full-stack development experience and UI/UX design work.

## Features

- Smooth scroll animations
- Responsive mobile navigation
- Project showcase with live links
- Technical skills display
- Education timeline
- Back-to-top button
- Modern dark theme

## Projects Showcased

- **SyncFil**: Operations management system (React, TypeScript, PostgreSQL)
- **Doktor Plaka**: Vinyl records e-commerce with PayMongo integration
- **T&T AniMotions**: Anime streaming platform
- **Sync-O**: Music streaming UI/UX design (Figma)

## Technologies

- HTML5
- CSS3
- JavaScript
- Intersection Observer API
- CSS Grid & Flexbox
- Google Fonts (Playfair Display, Inter)

## How to Run

1. Clone the repository
```bash
git clone https://github.com/tjyalmorin/portfolio-v2.git
```

2. Open `index.html` in your browser

No build process or dependencies required.
